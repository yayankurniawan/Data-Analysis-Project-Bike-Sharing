## Proyek Analisis Data: Bike Sharing Dataset
Proyek ini bertujuan untuk menganalisis Bike Sharing Dataset guna mengidentifikasi pola penggunaan sepeda serta faktor-faktor yang memengaruhi jumlah peminjaman. Hasil analisis diharapkan dapat menghasilkan wawasan dan informasi yang relevan sebagai dasar pengambilan keputusan berbasis data.

## Struktur Direktori

- dashboard/          : Berisi file untuk menjalankan dashboard Streamlit
- main_data.csv   : Dataset hasil pembersihan untuk visualisasi
- dashboard.py    : Script pembuatan dashboard interaktif

- data/               : Dataset mentah sebelum proses cleaning
- data_1.csv      : Data mentah pertama
- data_2.csv      : Data mentah kedua

- notebook.ipynb      : Proses analisis data, EDA, dan penarikan insight
- README.md           : Dokumentasi dan penjelasan proyek
- requirements.txt    : Daftar dependensi Python yang digunakan

## Tujuan Analisis

- Kapan jumlah peminjaman sepeda paling tinggi berdasarkan hari dalam seminggu?
- Bagaimana pengaruh cuaca terhadap jumlah peminjaman sepeda?
- Apakah terdapat perbedaan peminjaman antara hari kerja dan hari libur?
- Musim apa yang memiliki rata-rata peminjaman sepeda tertinggi?
- Jam berapa peminjaman sepeda paling tinggi?

## Cara Menjalankan Dashboard

```bash
streamlit run dashboard/dashboard.py
```
